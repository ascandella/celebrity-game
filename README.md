# Celebrity Game

Deployed at https://celebrityga.me/

This is a game I am writing to learn [Clojure](https://clojure.org/) and [React](https://reactjs.org/).

## Installation

Download from https://github.com/ascandella/celebrity-game.

## Docker Compose

```sh
docker-compose up -d
```

### Logs

```sh
docker-compose logs -f
```

### Stopping

```sh
docker-compose down
```
